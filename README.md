Deployment

This portfolio is deployed using **GitHub Pages**.

### To deploy updates:
1. Edit `index.html` locally  
2. Go to the repository  
3. Click **Add file → Upload files**  
4. Upload and **Commit changes**  
5. Wait 5–10 seconds  
6. Refresh: https://rahmanms.github.io
